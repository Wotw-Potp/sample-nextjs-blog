# Sample Next.js Blog

前にポートフォリオとして公開してたやつ

## Stacks
- docker
- Next.js(13.2.1)
- react(18.2.0)
- typescript(4.9.5)

## How-to
### Up
1. copy `.env.sample` to `.env`
2. fill all **Environment Variables** in `.env`
3. Start **Docker**
   ```sh
   $ sh up.sh
   ```

### Down
1. stop `yarn` watch mode on your shell (Ctrl + C)
2. Stop **Docker**
   ```sh
   $ sh down.sh
   ```
